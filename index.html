Features

Everything in the beta, free.

Two-way transfer

Drag files onto the window to upload. Drag out of the grid, or download to a folder you pick. Per-file progress with live MB/s.

Open & edit round-trip

Open a phone file in its Mac app. Save it — Droidwire pushes it back to the phone automatically.

Multiple devices

Two phones plugged in? Switch between them from the titlebar. USB and wireless at the same time.

Wireless ADB

Pair over Wi-Fi with a QR code (Android 11+). Transfer without a cable once paired.

Menu bar mode

Mini drop-zone in the menu bar for quick transfers without opening the full window.

Storage analyzer

Treemap of what's eating your phone storage. Drill into folders, clean up from the Mac.

Transfer queue

Pause, resume, reorder. Notification when the batch completes. Cancel mid-flight cleanly.

Previews & thumbnails

Image previews and grid thumbnails, including HEIC. Tabs, search, bookmarks for fast navigation.

Device tools

Screenshot capture, battery and storage detail, installed app list with APK export (ADB mode).

Install

Three minutes, start to first transfer.

Download and open the .dmg

Grab the latest Droidwire-x.x.x-arm64.dmg from the download button above, open it, and drag Droidwire into Applications.

First launch: bypass Gatekeeper once

Beta builds aren't signed with an Apple certificate yet, so macOS will warn you. Right-click Droidwire in Applications and choose Open, then confirm. You only do this once.

If macOS says the app is "damaged", run this in Terminal instead: xattr -dr com.apple.quarantine /Applications/Droidwire.app

Plug in your phone and choose a connection type

ADB (recommended): enable Developer Options (tap Build Number 7 times in Settings > About phone), turn on USB Debugging, plug in, and accept the "Allow USB debugging?" prompt on the phone. One-time per Mac.

Wi-Fi (Android 11+): same speed and features as ADB, no cable — pair once with a QR code, phone and Mac need to be on the same Wi-Fi network.

MTP (fallback, limited): zero setup — connect the cable and tap "File transfer" in the USB notification. No transfer progress bar, no APK install. Use this only if USB Debugging isn't an option.

Transfer

Your phone's files appear in the browser. Drag files in to upload, out to download. Downloads land in ~/Downloads/Droidwire/ by default (configurable).

Found a bug? This is a beta — that's the point. Grab the log via Droidwire menu > Show Logs in Finder and open an issue on GitHub with the log attached. Every report makes the app better.
